# BUA Health Outreach v2.3 — Premium Registration and Zendale Profile

## Registration improvements
- Keeps the one-question-per-screen structure.
- Reduces title size and vertical spacing so the question and options fit within the viewport more comfortably.
- Uses compact, responsive answer cards and two-column layouts for longer option sets.
- Keeps the answer area contained on very small screens instead of pushing navigation off-screen.
- Makes email mandatory in both client and server validation.
- Updates public copy to explain why email is required.

## Public experience redesign
- Replaces the previous homepage structure with a new institutional, editorial registration portal.
- Uses a full-height asymmetric composition and integrated outreach photography.
- Removes generic feature-card presentation.
- Adds a direct Zendale profile link in public footers.

## Zendale company profile
- Adds `/zendale`.
- Includes company overview, six service groups, healthcare network, reasons to choose Zendale, partnership call-to-action, and placeholder phone/email.
- Uses a separate premium visual identity led by Zendale navy and blue while remaining connected to the BUA outreach experience.
